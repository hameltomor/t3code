**1. Create `useImportWizardReducer.ts`:**

Define the wizard state machine:

```typescript
import type { HistoryImportConversationSummary, HistoryImportConversationPreview, HistoryImportExecuteResult, HistoryImportProvider, HistoryImportLinkMode, ProviderInteractionMode, RuntimeMode } from "@xbetools/contracts";

export type WizardStep = "provider-select" | "session-list" | "preview" | "options" | "result";

export interface ImportOptions {
  title: string;
  model: string;
  runtimeMode: RuntimeMode;
  interactionMode: ProviderInteractionMode;
  linkMode: HistoryImportLinkMode;
}

export interface WizardState {
  step: WizardStep;
  providerFilter: HistoryImportProvider | null;
  selectedSession: HistoryImportConversationSummary | null;
  preview: HistoryImportConversationPreview | null;
  importOptions: ImportOptions;
  result: HistoryImportExecuteResult | null;
  error: string | null;
}

export type WizardAction =
  | { type: "SET_PROVIDER_FILTER"; filter: HistoryImportProvider | null }
  | { type: "SELECT_SESSION"; session: HistoryImportConversationSummary }
  | { type: "SET_PREVIEW"; preview: HistoryImportConversationPreview }
  | { type: "GO_TO_OPTIONS" }
  | { type: "UPDATE_OPTIONS"; options: Partial<ImportOptions> }
  | { type: "SET_RESULT"; result: HistoryImportExecuteResult }
  | { type: "SET_ERROR"; error: string }
  | { type: "GO_BACK" }
  | { type: "RESET" };
```

Create `initialState` function that takes `defaultModel: string` and returns `WizardState` with step "provider-select", null selections, and sensible defaults for importOptions (runtimeMode: "full-access", interactionMode: "default", linkMode: "native-resume").

Create `wizardReducer(state: WizardState, action: WizardAction): WizardState` implementing:
- `SET_PROVIDER_FILTER`: update filter, stay on current step
- `SELECT_SESSION`: store session, advance to "session-list" (stays on session-list -- the preview step is navigated to by a separate mechanism when preview data loads)
- `SET_PREVIEW`: store preview, advance to "preview" step
- `GO_TO_OPTIONS`: advance to "options" step
- `UPDATE_OPTIONS`: merge partial options
- `SET_RESULT`: store result, advance to "result" step
- `SET_ERROR`: store error string
- `GO_BACK`: navigate to previous step (result->options, options->preview, preview->session-list, session-list->provider-select). Clear relevant state when going back (e.g., going back from preview clears preview and selectedSession)
- `RESET`: return initialState

Export a `useImportWizardReducer(defaultModel: string)` hook that wraps `useReducer(wizardReducer, initialState(defaultModel))`.
